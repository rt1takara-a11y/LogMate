# AI経営ログ（仮称） 仕様書

## 1. サービス概要

### コンセプト

**「毎日の経営ログをAIが学習し、オーナー専属の経営パートナーになるWebサービス」**

飲食店、美容院、花屋、カフェなどの小規模事業者向けに、日々の出来事や経営の悩み、スタッフの様子などを記録し、その内容をAIが分析・蓄積することで、経営アドバイスや振り返りを行うサービスを提供する。

既存の売上管理システムとは異なり、「経営者の頭の中」を資産として残すことを目的とする。

---

# 7. AIチャット

サービス最大の特徴。

AIはユーザーの過去ログを検索しながら回答する。

例

ユーザー

「最近売上が悪い」

AI

「過去30日のログを見ると、雨の日は売上が低い傾向があります。」

「また、新メニューの投稿が少なかった週も売上が下がっています。」

---

例

ユーザー

「スタッフ教育どうしたらいい？」

AI

「過去のログを見ると、○○さんは褒められた翌週に評価が向上しています。」

---

チャットでは以下を考慮する。

* 過去ログ
* スタッフ情報
* TODO
* 月次レポート
* 写真
* 売上

AIは必ず過去ログを引用しながら回答する。

# 17. サービスの価値

このサービスは「売上管理ツール」ではなく、「経営者の思考・経験・気づきを資産化するAI経営パートナー」である。

毎日の出来事や悩みを記録し、AIが長期的に学習・分析することで、一人で経営を抱え込むオーナーに寄り添い、客観的な視点と継続的なアドバイスを提供する。

最終的な目標は、「この店のことを一番理解している相談相手」としてAIが存在し、経営者が安心して相談できるパートナーとなることである。
